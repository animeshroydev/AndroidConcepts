What is Android Open Source and it's License?

BY ANIMESH ROY OCT 14 2017

Android is a open source software which means that the source code is freely available for everyone to use. There are few open source licenses use condition. The GPL(general public license) and Apache License. The GPL licenses allows "free distribution under the condition that further developements are put together in the same license".

Here comes the Apache License, which is more liberal and doesn't require modifications to be open and have the source code available.
