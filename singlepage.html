What is Android Open Source and it's License?

BY ANIMESH ROY

Android is a open source software which means that the source code is freely available for everyone to use. There are few open source licenses use condition. The GPL(general public license) and Apache License. The GPL licenses allows "free distribution under the condition that further developements are put together in the same license". Here comes the Apache License, which is more liberal and doesn't require modifications to be open and have the source code available. The most of the source code is released under the apache license. It means any one can change their code however they like. They must not required to make their code public. Thats why all the manufactures like samsung, Oppo, HTC, MI and others are implementing lots of innovative features in their device. And the source is hidden. That is I think the best ever license android provided to the manufactures. The linux kernel is used under the GPL(general public license) which means that the any changes made to the source code is released to public. This means that the manufacturers like Samsung, Oppo, Vivo, HTC and other android device manufacturers must release the source code to the public. Few popular and widely used licenses:-
1. Apache License 2.0
2. BSD 3-Clause "New" or "Revised" license
3. BSD 2-Clause "Simplified" or "FreeBSD" license
4. GNU General Public License (GPL)
5. GNU Library or "Lesser" General Public License (LGPL)
6. MIT license
7. Mozilla Public License 2.0
8. Common Development and Distribution License
9. Eclipse Public License.
